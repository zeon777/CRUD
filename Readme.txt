CRUD Приложение.
Используемые технологии:
*Maven
*Spring
*Hibernate
*Tomcat
Задача
1. создать CRUD (create, read, update, delete).
Есть  1 табличка User: id,name,age,isAdmin,createdDate
Необходимо реализовать стандартное CRUD приложение, которое отображает список
всех юзеров в базе (с пейджингом). С возможностью их удаления, редактирования,
добавления новых, и поиска по уже существующим.

Скрипт наполнения базы данных лежит в /script
Дэплой настроен в pom.xml
